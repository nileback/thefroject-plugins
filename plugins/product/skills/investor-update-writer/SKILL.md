# Investor Update Writer

Write investor updates that build trust and activate your investor base. Good updates are short, honest, and include a specific ask.

## Writes
- `outputs/investor-update-[period].md`

## Gather Inputs

Ask for:
1. **Period** — which month/quarter
2. **Key metrics** — MRR/ARR, growth rate, churn, runway, headcount
3. **Wins** — what went well (with specifics)
4. **Challenges** — what is hard right now (with context)
5. **Asks** — what you need from investors (intros, advice, hiring help)
6. **Upcoming** — what is planned for next period

## Structure

### Subject Line
"[Company Name] — [Month/Quarter] Update" (keep it simple, consistent)

### Metrics Dashboard (top of email)
Table format, 3-5 key metrics with month-over-month change:

| Metric | This Period | Last Period | Change |
|---|---|---|---|
| MRR | $X | $Y | +Z% |
| Customers | N | N | +N |
| Burn | $X/mo | $Y/mo | |
| Runway | N months | | |

### Highlights (3-5 bullets)
What went well. Be specific: "Closed [Company] at $X ARR" not "Strong sales month." Include numbers.

### Challenges (2-3 bullets)
What is hard. Be honest but frame constructively: "Churn increased to X%. Root cause is [specific]. We are addressing by [action]."

### Asks (1-3 items)
Specific, actionable requests:
- "Looking for intros to [role] candidates with [experience]"
- "Would value advice on [specific decision]"
- "Seeking intros to [specific company type] for partnerships"

Never: "Let us know if you can help." Always: "We need [specific thing]."

### Looking Ahead (2-3 sentences)
What is planned for next period. Set expectations.

## Writing Rules

- Under 500 words total
- Send consistently (same day each month/quarter)
- Metrics first, narrative second
- Bad news early, not buried
- One email, no attachments (investors skim on mobile)
