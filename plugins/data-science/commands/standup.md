# /standup — Generate Standup Update

Generate a standup update based on recent activity.

1. Check recent git commits (last 24 hours)
2. Check current branch and work in progress
3. Look at any TODO/FIXME comments added recently

Format as:
- **Yesterday:** What was completed
- **Today:** What's planned (based on context and open tasks)
- **Blockers:** Any issues identified in code or recent failures
