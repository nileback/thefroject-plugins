# /git-workflow — Git Operations

Help with common git workflows.

Available sub-commands:
- **branch** — Create a well-named feature branch
- **commit** — Stage and commit with conventional commit message
- **pr** — Create a pull request with description
- **sync** — Rebase current branch on latest main
- **cleanup** — Remove merged branches

Usage: `/git-workflow [branch|commit|pr|sync|cleanup]`
