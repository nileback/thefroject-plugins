# /pipeline-hiring: Hiring Pipeline

Run the full hiring pipeline from job description through to onboarding.

## Steps

### 1. Job description
Write an inclusive, clear job description with requirements, growth paths, and benefits. Match employer brand voice.

Output: job posting.

### 2. Interview design
Design structured interview rounds with scoring rubrics, competency questions, and evaluation criteria.

Output: interview guide.

### 3. Onboarding program
Build an onboarding and training program for new hires. Include first week schedule, key contacts, tools setup, and learning milestones.

Output: onboarding plan.

## How to use

Run: `/pipeline-hiring [role title]`
