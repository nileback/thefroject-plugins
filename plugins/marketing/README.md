# The Froject Marketing Plugin

Marketing workspace toolkit: content creation, campaigns, SEO, brand voice, launch playbooks, and analytics.

## Install

```
/plugin marketplace add bjorn-ingmanson/thefroject-plugins
/plugin install thefroject-marketing
```
