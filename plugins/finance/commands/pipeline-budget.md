# /pipeline-budget: Budget Cycle Pipeline

Run the full budget review and reporting pipeline.

## Steps

### 1. Budget analysis
Analyze current budget allocation, variances, and spending patterns. Flag items that need attention.

Output: variance analysis.

### 2. Forecast
Build forward-looking financial models with scenario analysis (best case, expected, worst case).

Output: forecast models.

### 3. Financial report
Generate formatted financial reports with key metrics, trends, and commentary.

Output: financial report.

### 4. Board package
Prepare an executive summary and board-ready financial package. Concise, visual, focused on decisions needed.

Output: board report.

## How to use

Run: `/pipeline-budget [period, e.g. "Q2 2026"]`
