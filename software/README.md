# The Froject Software Plugin

Software development toolkit: code review, TDD, debugging, security audits, refactoring, and CI/CD workflows.

## Install

```
/plugin marketplace add bjorn-ingmanson/thefroject-plugins
/plugin install thefroject-software
```
