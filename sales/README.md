# The Froject Sales Plugin

Sales workspace toolkit: outreach, deal strategy, competitor intel, pipeline management, and battlecards.

## Install

```
/plugin marketplace add bjorn-ingmanson/thefroject-plugins
/plugin install thefroject-sales
```
