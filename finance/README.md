# The Froject Finance Plugin

Finance workspace toolkit: budgeting, forecasting, financial reporting, billing ops, and audit preparation.

## Install

```
/plugin marketplace add bjorn-ingmanson/thefroject-plugins
/plugin install thefroject-finance
```
