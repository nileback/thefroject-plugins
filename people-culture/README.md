# The Froject People Culture Plugin

People & culture toolkit: job descriptions, interview design, performance reviews, DEI audits, and onboarding.

## Install

```
/plugin marketplace add bjorn-ingmanson/thefroject-plugins
/plugin install thefroject-people-culture
```
