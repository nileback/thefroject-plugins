# The Froject Product Plugin

Product management toolkit: PRDs, roadmaps, user research, metrics, launch planning, and stakeholder updates.

## Install

```
/plugin marketplace add bjorn-ingmanson/thefroject-plugins
/plugin install thefroject-product
```
