# The Froject Marketing Plugin

A marketing workspace toolkit for Claude Code. 12 skills, 5 commands, and 3 rules for content creation, campaigns, SEO, and analytics.

## Install

```
/plugin install thefroject-marketing
```

## Quick start

Try these after installing:

- `/draft blog post about customer retention strategies`
- `/campaign product launch for a SaaS analytics tool`
- `/audit https://yoursite.com`
- Ask Claude to "write a social media post about our new feature"
- Ask Claude to "build a brand voice guide from these three blog posts"

## Skills

- **Blog Writer** — Write engaging blog posts with SEO optimization
- **Email Writer** — Write effective emails for sales, marketing, and operations
- **Social Media Writer** — Create platform-specific social media content
- **Content Repurposer** — Transform one piece of content into multiple formats
- **Campaign Planner** — Plan marketing campaigns with messaging, channels, and timeline
- **Launch Playbook** — Generate a week-by-week launch plan covering pre-launch, launch day, and follow-up
- **Brand Voice Builder** — Generate a reusable brand voice guide from existing content samples
- **SEO Optimizer** — Optimize content and technical SEO for better rankings
- **Page CRO** — Audit and optimize web pages for conversion rate
- **Landing Page Writer** — Write high-converting landing page copy with clear structure
- **Competitor Researcher** — Research and analyze competitors systematically
- **Copy Analyzer** — Analyze existing copy and suggest concrete improvements with before/after examples

## Commands

- **/draft** — Quick-draft content (blog, email, social post, copy)
- **/campaign** — Plan a marketing or outreach campaign
- **/audit** — Quick marketing audit of a website or landing page
- **/launch-plan** — Generate a product or feature launch runbook
- **/pipeline-content** — Full content production pipeline: plan, research, draft, optimize, repurpose

## Hooks

The plugin includes three optional hooks that run after file edits:
- **Word count** — shows word count after writing
- **Markdown lint** — checks markdown formatting
- **Spell check** — catches typos in new files

Built from [The Froject](https://thefroject.com) template library.

The Froject is a guided wizard that generates complete Claude Code workspaces for any role. This plugin is a curated subset of our 153 skills, 47 commands, and 30 agents.
